# Best Friends Beach Game

A small static HTML, CSS and JavaScript game for GitHub Pages.

## Files

- `index.html`
- `style.css`
- `script.js`
- `assets/` with local SVG images, including beach characters and cheering fans

## How to run locally

Open `index.html` in a browser.

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to Settings.
4. Go to Pages.
5. Under Branch, choose `main` and `/root`.
6. Save.
7. Open the GitHub Pages URL after it is generated.

No build step is required.
